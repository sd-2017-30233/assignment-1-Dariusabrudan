In order to use the bank application, you have to install a IDE(Integrated development environment) for Java.
In order to access the application, you need to run the StartupWindow, which is consisted of the log in frame.
You can log in with a user that exists in the database, eighter an employee or the administrator.
If you log in as an employee, you have a set of operations and if you log in as administrator, you have other operations available.
